Research Overview

Investigating the power of headless language models in sample-efficient learning environments

  • Comparative analysis of headless vs. vanilla language models
  • Focus on Contrastive Weight Tying (CWT) pre-training objective
  • Evaluation using BabyLM Challenge datasets
  • Testing both MLM and GPT architectures
  • Assessment through BLiMP and GLUE benchmarks

Headless Language Modeling

Contrastive Weight Tying (CWT)

Key Innovation: Maintains alignment between input embeddings and output projections throughout training

  • Deviation from traditional MLM and Next Token Prediction approaches
  • Bidirectional mapping between word embeddings and output projections
  • Contrastive framework treating correct pairs as positive examples
  • Inspired by successful contrastive methods like SimCLR and CLIP

Training Setup

Evaluation Framework

BLiMP Benchmark

  • 67 linguistic phenomena paradigms
  • Minimal pairs methodology
  • Syntax, morphology, semantics
  • No task-specific fine-tuning
  • Raw linguistic knowledge assessment

GLUE Benchmark

  • 9 diverse NLU tasks
  • Sentiment analysis
  • Paraphrase detection
  • Natural language inference
  • Question answering

Complementary Assessment: BLiMP focuses on grammatical competence while GLUE evaluates downstream task performance

Research Contributions

Advancing understanding of efficient language model architectures

  • Architectural Innovation: First comprehensive evaluation of headless models on cognitively-inspired datasets
  • Sample Efficiency: Investigating learning effectiveness with limited, developmentally plausible data
  • Comparative Analysis: Systematic comparison across multiple model types and training configurations
  • Cognitive Relevance: Bridging computational linguistics and language acquisition research
  • Methodological Framework: Establishing evaluation protocols for sample-efficient pre-training
